# 🏛️ CivicAI

**CivicAI** is a GenAI-powered civic platform that simplifies how citizens interact with government services. Instead of navigating multiple websites and complex procedures, users can ask questions in natural language, discover relevant government schemes and services, understand document requirements, report civic issues, and track complaints through a single intelligent AI companion.

Designed with accessibility, transparency, and digital inclusion in mind, CivicAI makes everyday civic interactions faster, smarter, and more user-friendly through personalized assistance and multilingual support.

## ✨ Features

* 🤖 AI-powered civic assistant for government-related queries
* 🏛️ Smart government service and scheme discovery
* 📄 Document requirement guidance and application assistance
* 🚧 AI-assisted public issue reporting and categorization
* 📊 Complaint tracking with status updates
* 🌐 Multilingual support for better accessibility
* ♿ Simple, intuitive, and inclusive user experience

## 🛠️ Tech Stack

* React
* Node.js + Express
* MongoDB
* Google Gemini API
* Tailwind CSS

## 🔒 Security

* Input validation
* Secure API handling
* Environment variables for sensitive credentials
* Sanitized user input
